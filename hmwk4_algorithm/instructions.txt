 The 8-step recipe to approach programming challenges are:

        1. Understand the problem

        2. Work out examples

        3. Decide on a user interface

        4. Write down the algorithms (i.e., key components of the program)

        5. Look up information

        6. Write the program

        7. Run the program

        8. Verify the results

II. PALINDROME

Given a 3, 5 or 7-digit integer x, create a function that will return 
true if x is a palindrome integer and false otherwise.

Definition of Palindrome:
A word, phrase, or sequence that reads the same backward as forward,
e.g., madam or nurses run.

III. NUMBER IN LIST
